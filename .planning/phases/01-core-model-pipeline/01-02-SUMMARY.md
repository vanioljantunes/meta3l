## Accomplishments

- `meta3L()` implements the full pipeline: validate → filter NAs → rename to canonical → escalc → vcalc → rma.mv (dynamic formula) → robust CR2 → compute_i2 → back-transform → structure S3 object
- All 6 measure types supported: PLO, PAS, SMD, MD, RR, OR
- PFT and unsupported measures rejected early with informative error (via resolve_transf)
- NA rows in required columns warned and filtered before analysis
- Custom cluster column support via dynamic rma.mv formula
- `meta3l_result` S3 object contains: model (robust.rma), data, V, i2, transf, measure, cluster, rho, slab, TE_id, estimate, ci.lb, ci.ub
- `print.meta3l_result()`: compact display with pooled estimate, CI, k, n, rho, I2 breakdown, robust SE note
- `summary.meta3l_result()`: print + sigma2 components, robust variance table (estimate, SE, z, p, CI on log scale), convergence info
- R CMD check --as-cran: **0 errors, 0 warnings**, 2 acceptable NOTEs (new submission, time verification)
