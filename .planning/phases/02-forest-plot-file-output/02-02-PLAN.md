Create `meta3l/R/forest.meta3L.R` â€” the S3 method. Follow the grid layout architecture from RESEARCH.md Pattern 1-5.

**Function signature** (per CONTEXT.md locked decisions):
```r
#' @export
forest.meta3L <- function(x,
                          ilab        = NULL,
                          ilab.lab    = NULL,
                          sortvar     = NULL,
                          refline     = NULL,
                          xlim        = NULL,
                          at          = NULL,
                          xlab        = NULL,
                          showweights = TRUE,
                          colshade    = rgb(0.92, 0.92, 0.92),
                          squaresize  = 1,
                          file        = character(0),
                          format      = "png",
                          width       = NULL,
                          height      = NULL,
                          ...)
```

**Implementation structure (sequential steps inside the function):**

1. **Validate input:** `stopifnot(inherits(x, "meta3l_result"))`

2. **Extract and back-transform data:**
   - `yi_bt <- x$transf(x$data$yi)`
   - `ci_lb_bt <- x$transf(x$data$yi - stats::qnorm(0.975) * sqrt(x$data$vi))`
   - `ci_ub_bt <- x$transf(x$data$yi + stats::qnorm(0.975) * sqrt(x$data$vi))`
   - `slab_vals <- x$data[[x$slab]]`
   - `n_studies <- nrow(x$data)`
   - Weights: `w <- 1 / sqrt(x$data$vi); w_norm <- (w / max(w, na.rm = TRUE)) * squaresize`

3. **Sort if requested:** If `sortvar` is non-NULL, reorder all vectors by `x$data[[sortvar]]`

4. **Compute layout parameters:**
   - `xlim_final <- if (!is.null(xlim)) xlim else auto_xlim(x$measure, yi_bt, ci_lb_bt, ci_ub_bt)`
   - `refline_final <- if (!is.null(refline)) refline else auto_refline(x$measure)`
   - `mlab <- format_mlab(x$i2)`
   - Number of ilab columns: `n_ilab <- if (!is.null(ilab)) length(ilab) else 0L`
   - ilab labels: `ilab_labels <- if (!is.null(ilab.lab)) ilab.lab else ilab`

5. **Compute text for right panel:**
   - Per-study: `sprintf("%.2f [%.2f; %.2f]", yi_bt[i], ci_lb_bt[i], ci_ub_bt[i])`
   - Pooled: `sprintf("%.2f [%.2f; %.2f]", x$estimate, x$ci.lb, x$ci.ub)`
   - For comparison measures (SMD, MD, RR, OR): append `sprintf("; p = %.4f", x$model$pval)` to pooled text
   - Weight text: `sprintf("%.1f%%", (w / sum(w)) * 100)` (percentage weights)

6. **Resolve file path and open device:**
   - `out_file <- resolve_file(x, file, format)`
   - `dims <- auto_dims(n_studies, width, height)`
   - If `out_file` is not NULL: open device with `if (format == "pdf") grDevices::pdf(out_file, width = dims$width / 300, height = dims$height / 300) else grDevices::png(out_file, width = dims$width, height = dims$height, res = 300L)`
   - `on.exit(grDevices::dev.off(), add = TRUE)` immediately after opening
   - If `out_file` is NULL: open `grDevices::pdf(nullfile())` for drawing (will display in RStudio if available)

7. **Build grid layout:**
   - `grid::grid.newpage()`
   - Define rows: header row (1) + study rows (n_studies) + blank row (1) + pooled row (1) + mlab row (1) + axis row (1) = total_rows
   - Define columns: studlab | [ilab_1 ... ilab_n] | [weight] | gap | CI_panel | gap | results_text
   - Use `grid::stringWidth()` inside a temporary device context to measure column widths (measure longest string in each column, convert to cm)
   - `grid::pushViewport(grid::viewport(layout = grid::grid.layout(nrow = total_rows, ncol = n_cols, widths = col_widths, heights = grid::unit(rep(1, total_rows), "lines"))))`

8. **Draw header row:** Study label header, ilab headers, weight header (if showweights), results header â€” all via `grid::grid.text()` in appropriate cells with `gp = grid::gpar(fontface = "bold", cex = 0.75)`

9. **Draw study rows (loop):**
   For each study i (rows 2 to n_studies+1):
   a. Zebra shading: if i is even, push viewport for full row span across all columns, call `draw_zebra_rect(colshade)`, pop
   b. Study label: push viewport (row, studlab_col), `grid::grid.text(slab_vals[i], x=unit(0, "npc"), just="left", gp=gpar(cex=0.75))`, pop
   c. ilab columns: for each ilab col j, push viewport, `grid::grid.text(as.character(x$data[[ilab[j]]][i]), ...)`, pop
   d. Weight column (if showweights): push viewport, `grid::grid.text(weight_text[i], ...)`, pop
   e. CI panel: push viewport with `xscale = xlim_final`, call `draw_ci_line(ci_lb_bt[i], ci_ub_bt[i])`, call `draw_square(yi_bt[i], w_norm[i] * 0.8)` (0.8 = max fraction of line height), handle truncation (if ci extends beyond xlim, draw arrow), pop
   f. Results text: push viewport, `grid::grid.text(result_text[i], x=unit(0, "npc"), just="left", gp=gpar(cex=0.75))`, pop

10. **Draw pooled row:**
    - Push CI panel viewport for pooled row with `xscale = xlim_final`
    - Call `draw_diamond(x$ci.lb, x$estimate, x$ci.ub)` (already back-transformed)
    - Pop viewport
    - Draw "Random effects model" label in studlab column
    - Draw pooled estimate text (with p-value for comparison measures) in results column

11. **Draw mlab row:** Push viewport spanning studlab through last ilab column, draw `mlab` text left-justified with `gp = gpar(cex = 0.65, fontface = "italic")`

12. **Draw reference line:** If refline_final is not NULL, push CI panel viewport spanning all study rows + pooled row, draw vertical `grid::grid.segments` at refline_final with `lty = "dashed"`

13. **Draw axis:** Push CI panel viewport for axis row with `xscale = xlim_final`, call `grid::grid.xaxis(at = if (!is.null(at)) at else pretty(xlim_final, n = 5), gp = gpar(cex = 0.65))`. If `xlab` is non-NULL, draw it below axis.

14. **Cleanup:** `grid::popViewport()` for the root layout viewport. `on.exit` handles device closure.

15. **Return:** `invisible(out_file)` â€” the path to the saved file (or NULL if display only).

**Important implementation notes:**
- R >= 4.0 compat: NO `|>` or `\(x)` anywhere
- Use `grid::` prefix for ALL grid calls (the package uses explicit namespacing, not @import grid)
- For PDF dimensions: convert pixels to inches by dividing by 300 (our internal unit is pixels at 300 dpi)
- Zebra shading must span the FULL width of the layout (all columns), not just the CI panel
- The refline must be drawn BEHIND study data (draw it before squares/diamonds, or use a separate viewport layer)
- For CIs extending beyond xlim: clip the line at xlim boundary and draw a small arrow (grid::grid.segments with arrow = grid::arrow(length = unit(0.05, "inches")))
