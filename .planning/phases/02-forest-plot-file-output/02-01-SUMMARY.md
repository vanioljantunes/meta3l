## Decisions Made

- `character(0)` is the sentinel for auto-name (as opposed to `NULL` for display-only or a string for explicit path) — enables `missing()` pattern in the calling forest function
- `draw_*` functions are intentionally viewport-context-free — they assume the caller has pushed an appropriate viewport with `xscale`. This matches the architecture of `meta::forest.meta` and gives Plan 02 full layout control
- `normalizePath(mustWork=TRUE)` is safe inside `read_multisheet_excel()` because `readxl::excel_sheets(path)` has already validated the file exists
- `auto_xlim` for SMD/MD uses symmetric range around zero; for RR/OR uses exp-scale range with positive lower bound (minimum 0.001) — aligns with how these measures are displayed in publication plots
